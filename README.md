# Password Generator

# Purpose
A website that randomly generates a password based on user selected criteria. 
* User selected password criteria can include:
    * 8 - 128 characters
    * numbers
    * lowercase letters
    * uppercase letters
    * special characters

# Built With
* HTML
* CSS
* JavaScript
